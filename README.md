# Pagos Yape & Plin

Página web simple para recibir pagos mediante Yape y Plin.

## Número de pago
935128270

## Cómo usar

1. Subir todos los archivos a un repositorio en GitHub.
2. Activar GitHub Pages en Settings > Pages.
3. Seleccionar la rama main y guardar.
4. Tu página estará disponible en:
   https://TUUSUARIO.github.io/NOMBRE-REPO/

Incluye:
- Código QR
- Botón WhatsApp
- Campo para código de aprobación Yape
